# Análise de Churn de Clientes

## Descrição

Este projeto tem como objetivo visualizar a distribuição de clientes que cancelaram (churn) e os que permaneceram ativos, utilizando gráficos de pizza com a biblioteca Matplotlib.

A análise de churn é essencial para empresas que desejam entender a retenção de clientes e tomar decisões estratégicas baseadas em dados.

---

## Tecnologias utilizadas

* Python
* Pandas
* Matplotlib

---

## Estrutura dos dados

O dataset utilizado contém uma coluna chamada:

* **Churn**

  * `1` → Cliente cancelou (Sim)
  * `0` → Cliente permaneceu (Não)

---

## Visualização

O gráfico gerado é um gráfico de pizza que representa:

* Percentual de clientes que deram churn
* Percentual de clientes ativos

### Características do gráfico:

* Cores diferenciadas para cada categoria
* Destaque visual para clientes que cancelaram
* Exibição de porcentagens
* Layout centralizado e legível
